steps to write I & D xpath
1. inspect Independent element (IE)--> and note down its html code
2. place the mouse pointer on html code of IE and move in upward direction 
   till it highlights both IE & DE (common parent)
3. note down the path from IE to CP
4. use down arrow key to navigate from CP to DE , note down its path
5. write xpath from IE--> CP--> DE
